# Notion Blog Template

## Getting Started

### Create env.local

```
NOTION_SECRET=
NOTION_DATABASE=
NEXT_PUBLIC_BASE_URL=
```

### Install dependencies

```bash
pnpm install
```

### Run Server

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Run Build

```bash
pnpm start
```
